# 全平台配图策略与自动化方案

## 一、平台配图规范总览

### 1. 小红书 (Xiaohongshu)

| 属性 | 要求 |
|------|------|
| **推荐尺寸** | 1080×1080px (1:1) 或 1080×1440px (3:4) |
| **格式** | JPG, PNG |
| **图片数量** | 1-18张,推荐3-6张 |
| **文件大小** | 单张≤5MB |

**热门AI帖配图风格**:
- 封面图:渐变背景+大标题+emoji装饰
- 信息图:数据对比表格,柱状图
- 卡片图:要点列表,数字编号
- 总结图:结论建议,场景推荐

**设计要点**:
- 高饱和度配色吸引眼球
- 大字体标题,信息密度适中
- emoji点缀增加亲和力
- 统一视觉风格形成系列

---

### 2. 微博 (Weibo)

| 属性 | 要求 |
|------|------|
| **推荐尺寸** | 1080×1080px (九宫格) / 1920×1080px (横图) |
| **格式** | JPG, PNG, GIF |
| **图片数量** | 1-9张 |
| **文件大小** | 单张≤5MB |

**热门AI帖配图风格**:
- 九宫格:信息量大,完整故事线
- 长图:教程/攻略类,竖版滚动
- 表情包+文字:轻松调侃风格
- 对比图:前后对比,产品对比

**设计要点**:
- 第一张图最关键(决定点击率)
- 九图要有叙事性
- 适当使用梗图增加传播

---

### 3. 知乎 (Zhihu)

| 属性 | 要求 |
|------|------|
| **封面尺寸** | 1120×630px (16:9) |
| **正文配图** | 宽度≥1080px,高度不限 |
| **格式** | JPG, PNG, GIF |
| **图片数量** | 不限,推荐5-15张 |

**热门AI帖配图风格**:
- 架构图:系统架构,流程图
- 数据图:性能对比,benchmark
- 思维导图:知识点总结
- 代码截图:带语法高亮

**设计要点**:
- 专业严谨,数据可视化
- 高信息密度,干货为主
- 封面图要有吸引力
- 图文配合,逻辑清晰

---

### 4. B站 (Bilibili)

| 属性 | 要求 |
|------|------|
| **动态配图** | 1080×1440px (3:4) 或 1080×1080px (1:1) |
| **视频封面** | 1920×1080px (16:9) |
| **专栏封面** | 1146×717px |
| **格式** | JPG, PNG |
| **文件大小** | ≤5MB |

**热门AI内容配图风格**:
- 动态:截图+文字说明
- 视频:醒目封面+关键信息
- 专栏:配图+代码+架构图

**设计要点**:
- 二次元友好风格
- 视频封面要有冲击力
- 技术内容可多用截图

---

### 5. 抖音 (Douyin)

| 属性 | 要求 |
|------|------|
| **推荐尺寸** | 1080×1920px (9:16 竖屏) |
| **替代尺寸** | 1080×1440px (3:4) |
| **格式** | JPG, PNG |
| **图片数量** | 2-35张 |
| **文件大小** | 单张≤5MB |

**热门AI内容配图风格**:
- 竖屏信息图:从上到下阅读流
- 对比图:使用前/使用后
- 卡片滑动:要点逐一展示

**设计要点**:
- 重要内容放中上部
- 顶部底部留安全区
- 大字体,高对比度
- 适合竖屏浏览

## 三、图片类型与模板

### Type 1: 封面图 (Cover)
- **用途**: 帖子主视觉,吸引点击
- **元素**: 渐变背景 + 主标题 + 副标题 + 装饰
- **变体**:
  - 渐变风格(科技感)
  - 纯色风格(简约)
  - 图片+蒙版(真实感)

### Type 2: 信息图 (Infographic)
- **用途**: 数据展示,对比分析
- **元素**: 表格/图表 + 标签 + 数据
- **变体**:
  - 对比表格
  - 柱状图
  - 雷达图
  - 时间线

### Type 3: 卡片图 (Card)
- **用途**: 要点列举,功能介绍
- **元素**: 编号/图标 + 标题 + 描述
- **变体**:
  - 垂直列表
  - 横向卡片
  - 网格布局

### Type 4: 总结图 (Summary)
- **用途**: 结论,建议,行动指南
- **元素**: 场景 → 推荐方案
- **变体**:
  - 场景映射
  - 步骤流程
  - 检查清单
